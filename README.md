# Persona-judge: Personalized Alignment of Large Language Models via Token-level Self-judgment
This repository contains the official implementation of the paper Persona-judge: Personalized Alignment of Large Language Models via Token-level Self-judgment (https://arxiv.org/abs/2504.12663). More details of the code will be released soon, stay tuned!
